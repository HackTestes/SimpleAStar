Introduction

What is this site?

This site is a front end to a C++ application running either locally or in the cloud. This browser application helps you to use and visualize the results.

How do I use it?

You fill the form containing the parameters necessary for the C++ app to create a json input. After receiving the result, you can display the result.

How do I run the complete application locally?

Just download the script from here and run it in the terminal on a Linux operation system.

sh AStarInstallScript.sh

Does it have a GitHub page?

Yes, you can find it here.
